# Video Stabilizer

Video Stabilization algorithm implementation using OpenCV

## Table of Contents

- [Installation](#installation)
- [Usage](#usage)
- [Examples](#Examples)

## Installation

```sh
git clone https://github.com/AdiAlbum1/video_stabilizer
cd video_stabilizer
pip install -r requirments.txt
```

## Usage

```sh
python stabilize_video.py
```
